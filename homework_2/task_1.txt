Операторы ++ и -- могут быть расположены не только после, но и до переменной.
1) Когда оператор идет после переменной - это «постфиксная форма»: a++.
2) «Префиксная форма» — это когда оператор идёт перед переменной: ++a.
Обе эти конструкции делают одно и то же: увеличивают a на 1.
сть ли разница между ними? Да, но увидеть её мы сможем, только если будем использовать значение,
которое возвращают ++/--. Префиксная форма возвращает новое значение, в то время как постфиксная
форма возвращает старое (до увеличения/уменьшения числа).

Рассмотрим код д.з.:

var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1

Видим, что c = 2. Все правильно, так как использовали префиксную форму, поэтому возвращаем новое значение.
В случае с переменной d, получаем 1, потому что постфиксная форма, которая возвращает старое значение.

c = (2+ ++a); alert(c);      // 5 // а уже увеличили на 1 => новое значение 5
d = (2+ b++); alert(d);      // 4 // тут наоборот старое
alert(a);                    // 3
alert(b);                    // 3
